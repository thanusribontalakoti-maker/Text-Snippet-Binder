# 📋 Snippet Binder

Snippet Binder is a modern web application inspired by Pastebin that allows users to store, organize, search, and manage reusable code snippets and notes. The application works entirely in the browser using Local Storage.

---

## ✨ Features

- ➕ Create new snippets
- 📂 Organize snippets by category
- 🔍 Search snippets instantly
- 🏷️ Category filtering
- 📋 One-click Copy to Clipboard
- 🗑️ Delete snippets
- 📅 Timestamp for every snippet
- 💾 Local Storage support
- 📱 Responsive Design
- 🎨 Modern Glassmorphism UI

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage API
- Clipboard API

## 🚀 How to Run

1. Clone the repository

```bash
git clone https://github.com/yourusername/Snippet-Binder.git
```

2. Open the project folder.

3. Run **index.html** using your browser or Live Server.

---

## 💡 How It Works

Users can:

- Create snippets with title, category, and content.
- Save snippets using Local Storage.
- Browse all snippets.
- Search snippets instantly.
- Filter by category.
- Copy snippets to clipboard.
- Delete unwanted snippets.
